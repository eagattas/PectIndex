# PectIndex
EECS 498-010 Fall 2017 Senior Design Project

Team Members:
	Evan Agattas
	Evan Brunt
	Dillon Kesto
	Trevor Rees

PectIndex is a desktop application for doctors to analyze scans of patients with Pectus Excavatum deformities. It allows the doctor to upload a scan of a patient, render the image, and select a slice or range of slices to analyze for different medical indexes.
